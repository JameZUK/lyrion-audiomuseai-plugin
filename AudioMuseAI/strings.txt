PLUGIN_AUDIOMUSEAI
	EN	AudioMuse-AI

PLUGIN_AUDIOMUSEAI_DESC
	EN	Browse and queue tracks using AudioMuse-AI's sonic similarity, sonic fingerprint, instant playlists, song alchemy, and pathfinding. Adds a top-level entry under My Music.

PLUGIN_AUDIOMUSEAI_URL
	EN	AudioMuse-AI server URL

PLUGIN_AUDIOMUSEAI_URL_DESC
	EN	Base URL of your AudioMuse-AI Flask container (e.g. http://localhost:8000 or http://my-host:8000). Do not include /api.

PLUGIN_AUDIOMUSEAI_TOKEN
	EN	API token

PLUGIN_AUDIOMUSEAI_TOKEN_DESC
	EN	Bearer token for AudioMuse-AI. Generate one in the AudioMuse web UI under Administration. Leave blank if AUTH_ENABLED is false on the server.

PLUGIN_AUDIOMUSEAI_DEFAULT_COUNT
	EN	Default number of tracks to return

PLUGIN_AUDIOMUSEAI_DEFAULT_COUNT_DESC
	EN	How many tracks AudioMuse should return for similarity, fingerprint, instant-playlist, alchemy, and mood queries. Range 5–100.

PLUGIN_AUDIOMUSEAI_DSTM
	EN	Auto-extend the queue (Don't Stop The Music)

PLUGIN_AUDIOMUSEAI_DSTM_DESC
	EN	When the player's queue runs low, automatically add more similar / fingerprint / alchemy tracks based on the most recently activated dynamic mode for that player. Use the Dynamic Playlists submenu to start a mode; "Stop auto-extend" clears it.

PLUGIN_AUDIOMUSEAI_CONNECTION_OK

PLUGIN_AUDIOMUSEAI_CONNECTION_FAIL
