PLUGIN_AUDIOMUSEAI
	EN	AudioMuse-AI

PLUGIN_AUDIOMUSEAI_DESC
	EN	Browse and queue tracks using AudioMuse-AI's sonic similarity, sonic fingerprint, instant playlists, song alchemy, and pathfinding. Adds a top-level entry under My Music.

PLUGIN_AUDIOMUSEAI_URL
	EN	AudioMuse-AI server URL

PLUGIN_AUDIOMUSEAI_URL_DESC
	EN	Base URL of your AudioMuse-AI Flask container (e.g. http://localhost:8000 or http://my-host:8000). Do not include /api. Will auto-prefix http:// if you omit the scheme.

PLUGIN_AUDIOMUSEAI_TOKEN
	EN	API token

PLUGIN_AUDIOMUSEAI_TOKEN_DESC
	EN	Bearer token for AudioMuse-AI. Generate one in the AudioMuse web UI under Administration. Leave blank if AUTH_ENABLED is false on the server. Surrounding whitespace is trimmed automatically.

PLUGIN_AUDIOMUSEAI_DEFAULT_COUNT
	EN	Default number of tracks to return

PLUGIN_AUDIOMUSEAI_DEFAULT_COUNT_DESC
	EN	How many tracks AudioMuse should return for similarity, fingerprint, instant-playlist, alchemy, and mood queries. Range 5–100, clamped automatically.

PLUGIN_AUDIOMUSEAI_DSTM
	EN	Auto-extend the queue (Don't Stop The Music)

PLUGIN_AUDIOMUSEAI_DSTM_DESC
	EN	When the player's queue runs low, automatically add more similar / fingerprint tracks based on the most recently activated dynamic mode for that player. Use the Dynamic Playlists submenu to start a mode; "Stop auto-extend" clears it.

PLUGIN_AUDIOMUSEAI_NAME_FORMAT
	EN	Auto-name format for "Save current queue"

PLUGIN_AUDIOMUSEAI_NAME_FORMAT_DESC
	EN	How the auto-named playlist is titled when you tap "Save current queue as playlist". Web UI users can still type a custom name in the field; this setting only controls the default. No AI is used — names are derived from local data.

PLUGIN_AUDIOMUSEAI_NAME_FORMAT_TIMESTAMP
	EN	Timestamp — e.g. "AudioMuse 2026-05-03 14:30"

PLUGIN_AUDIOMUSEAI_NAME_FORMAT_FIRST_TRACK
	EN	First track — e.g. "AudioMuse: 1933 - Frank Turner"

PLUGIN_AUDIOMUSEAI_NAME_FORMAT_ARTIST_MIX
	EN	Artist mix — e.g. "AudioMuse: Frank Turner, Phish, U2 (24t)"

PLUGIN_AUDIOMUSEAI_NAME_FORMAT_MOOD_TAGGED
	EN	Mood-tagged — e.g. "AudioMuse: Calm / Ambient - 2026-05-03 14:30"

PLUGIN_AUDIOMUSEAI_NAME_FORMAT_PROMPT
	EN	Last Instant Playlist prompt — e.g. "AudioMuse: low-energy ambient"

PLUGIN_AUDIOMUSEAI_AUTO_SAVE_INSTANT
	EN	Auto-save Instant Playlist results

PLUGIN_AUDIOMUSEAI_AUTO_SAVE_INSTANT_DESC
	EN	After every Instant Playlist (text-prompt) action, save the queue as a Lyrion playlist named after the prompt — e.g. "AudioMuse: low-energy ambient".

PLUGIN_AUDIOMUSEAI_AUTO_SAVE_MOOD
	EN	Auto-save Browse-by-Mood results

PLUGIN_AUDIOMUSEAI_AUTO_SAVE_MOOD_DESC
	EN	After tapping a Browse-by-Mood preset, save the queue as a Lyrion playlist named after the mood and current time — e.g. "AudioMuse: Calm / Ambient - 2026-05-03 14:30".
